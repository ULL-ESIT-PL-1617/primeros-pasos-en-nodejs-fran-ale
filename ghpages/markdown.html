Markdown

Markdown

Markdown es un lenguaje de marcado ligero creado por John Gruber que trata de conseguir la máxima legibilidad y facilidad de publicación tanto en su forma de entrada como de salida.

Cabeceras:

# Cabecera 1
## Cabecera 2
### Cabecera 3  
Y así sucesivamente.

Cabecera 1

Cabecera 2

Cabecera 3

Titulos: Se usan poniendo almenos 2 simbolos '=' en la linea siguiente

  Titulo
  ==

Titulo

Tipos de texto: italic, negrita, and codigo().

*italic*,  **negrita**, and `codigo()`.

Citas: Puedes citar textos y anidarlos con '>'

> cita
>> cita2

cita

cita2

Listas: Se pueden hacer listas con '*','-','+' o enumeradas '1.','2.'

1. e1
- e2
+ e3
  1. a
  2. b
  • c
  • d

Codigo: Tambien se puede hacer cógido multilinea con ~~~ al principio y al final.

~~~
if (code == true &&) {
    func();
}
~~~ 

if (code == true &&) {
    func();
}
